Add `@rawdash/connector-clickup`, a new connector that syncs ClickUp spaces, folders, lists, and tasks (with status, priority, assignees, tags, and lifecycle timestamps) plus derived task lifecycle events (created / closed) from a ClickUp workspace. Authenticated via a ClickUp personal API token. Drives open-work stats, created/closed throughput timeseries, and status / list distributions.
