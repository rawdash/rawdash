Add `@rawdash/connector-branch`, a new connector that syncs daily Branch attribution metrics (installs, opens, conversions, and estimated cost by channel and campaign) and aggregated deep-link click events (by channel, campaign, feature) from the Branch Cross-Platform Analytics API. Authenticated via Branch app key + secret.
