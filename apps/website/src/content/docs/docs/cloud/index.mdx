**Rawdash Cloud** is the managed version of Rawdash. You configure your
connectors through a web UI or API, and Rawdash Cloud handles:

- Automatic background syncs on a managed schedule
- Persistent storage with automatic backups
- A hosted dashboard UI (no frontend code required)
- HTTPS and CDN delivery
- Team access controls (coming soon)

No servers to run. No infra to maintain.

## How it relates to the OSS version

Rawdash Cloud runs the same core sync engine as the open-source version. Your
connector configurations are portable — export them from your OSS setup and
import them into Cloud, or vice versa.

| | OSS | Cloud |
|--|-----|-------|
| Self-hosted | ✅ | ❌ |
| Managed syncs | ❌ | ✅ |
| Hosted UI | ❌ | ✅ |
| Custom connectors | ✅ | Coming soon |
| MCP access | ✅ | ✅ |
| Open source | ✅ | ❌ |
