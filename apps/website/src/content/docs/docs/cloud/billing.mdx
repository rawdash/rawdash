| Dashboards | 1 | 5 | Unlimited |

| Team members | 1 | 3 | Unlimited |
| MCP access | ✅ | ✅ | ✅ |

## How usage is measured

Rawdash Cloud bills on **sync operations**:

- Each connector sync counts as one operation
- Failed syncs do not count
- Manual syncs triggered via the API count the same as scheduled syncs

## Data retention

Data older than your plan's retention window is automatically pruned. If you
downgrade to a lower-retention plan, older data is removed on the next
scheduled pruning run (up to 24 hours).

## Cancellation and data export

You can cancel your plan at any time from **Settings → Billing**. After
cancellation, your data remains accessible for 30 days. Before the 30-day
window closes, you can export a full JSON dump of all synced data from the
**Settings → Export** page.
