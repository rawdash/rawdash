Rawdash ships 12 built-in connectors. Each syncs data from a third-party API into the storage engine, where your widgets query it.

## Engineering

- [GitHub](/docs/connectors/github-actions/) - Sync pull requests, issues, deployments, releases, CI runs, and contributor activity from a GitHub repository.
- [Sentry](/docs/connectors/sentry/) - Sync issues, issue events, releases, and hourly error rates from a Sentry organization.

## Product

- [Jira](/docs/connectors/jira/) - Sync projects, users, sprints, issues, and issue status-change events from a Jira Cloud site.
- [Linear](/docs/connectors/linear/) - Sync teams, members, cycles, issues, and issue state-transition events from a Linear workspace.
- [PostHog](/docs/connectors/posthog/) - Sync feature flags, per-day event volume, feature flag usage, active users, and funnel conversion from a PostHog project.

## Analytics

- [Google Analytics](/docs/connectors/google-analytics/) - Sync daily GA4 traffic, acquisition, top pages, events, conversions, and geography metrics from a Google Analytics 4 property.
- [Mixpanel](/docs/connectors/mixpanel/) - Sync Mixpanel active-user counts, per-event volume, funnel conversion, and cohort retention as metric time series.

## Sales

- [HubSpot](/docs/connectors/hubspot/) - Sync CRM contacts, companies, and deals plus deal stage-change events and marketing email campaign stats from HubSpot.

## Finance

- [AWS Cost Explorer](/docs/connectors/aws-cost/) - Track AWS spend over time and projected month-end costs, optionally broken down by service, account, tag, or cost category.
- [Stripe](/docs/connectors/stripe/) - Sync customers, products, prices, subscriptions, and invoices alongside charge, payment, dispute, and refund events from your Stripe account.

## Infrastructure

- [AWS CloudWatch](/docs/connectors/aws-cloudwatch/) - Pull declared CloudWatch metric time series (any namespace, statistic, and period) into a single metric series per query.
- [Vercel](/docs/connectors/vercel/) - Sync Vercel projects and deployments - including build state, target, git ref, and build duration - across your team.
