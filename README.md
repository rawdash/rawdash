# Rawdash

**An open-source headless dashboard backend for any team.**

Rawdash is a framework-agnostic TypeScript library you embed into your existing backend to handle the data side of dashboards as a solved problem: pre-built connectors for SaaS tools, a background sync engine, a pluggable storage layer, and a clean API that any frontend can consume.

Bring your own frontend. Run it anywhere.
