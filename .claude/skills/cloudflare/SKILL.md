# Cloudflare

CLI for managing Cloudflare resources (Workers, Pages, D1, KV, R2, DNS, Workers Secrets, and more) via `wrangler`. Run `wrangler --help` for available commands.

API token is available as `CLOUDFLARE_API_TOKEN` environment variable — wrangler picks it up automatically, no `wrangler login` needed.
